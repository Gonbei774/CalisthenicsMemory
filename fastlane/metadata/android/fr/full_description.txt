Un tracker simple d'entraînement au poids du corps que vous pouvez personnaliser selon vos besoins.

Créez vos propres exercices, organisez-les en groupes et suivez votre progression à votre manière. Pas de modèles rigides - adaptez l'application à votre méthode d'entraînement.

FONCTIONNALITÉS

Personnalisation Complète
• Créez n'importe quel exercice avec n'importe quel nom
• Organisez les exercices en groupes personnalisés
• Choisissez un suivi basé sur les répétitions ou le temps
• Définissez des progressions sur 10 niveaux adaptées à votre méthode
• Support pour les exercices unilatéraux (un bras/une jambe)
• Exercices favoris pour un accès rapide avec l'icône étoile (★)

Deux Modes d'Entraînement
• Mode Enregistrement : Enregistrez rapidement vos entraînements après l'effort
• Mode Entraînement : Entraînement auto-guidé avec minuteurs intégrés pour le compte à rebours, l'exécution et les intervalles de repos

Enregistrements d'Entraînement
• Enregistrez la date, l'heure, les séries et les répétitions/secondes
• Ajoutez des commentaires pour suivre les notes de forme et les observations
• Modifiez ou supprimez les enregistrements passés

Suivi de Progression
• Vue liste : Parcourez toutes les sessions d'entraînement
• Vue graphique : Visualisez la progression avec les statistiques moyennes/max/totales
• Vue objectifs : Suivez l'atteinte des objectifs

Système d'Objectifs
• Définissez des objectifs comme "3 séries × 50 répétitions"
• Indicateurs de progression codés par couleur

Gestion des Données
• Exporter/importer au format JSON
• Vos données restent sur votre appareil
• Aucune connexion Internet requise

Que vous suiviez Convict Conditioning, StartBodyweight ou votre propre programme personnalisé, cette application s'adapte à votre philosophie d'entraînement.

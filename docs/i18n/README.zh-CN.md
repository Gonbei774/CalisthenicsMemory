## 关于此应用

Calisthenics Memory是一款Android应用，用于记录和管理俯卧撑和深蹲等自重训练（徒手健身）。自由创建动作，按组织分组，以您的方式记录进度。

### 特点

- **完全可自定义** - 自由创建动作，按分组组织，10级别管理，收藏功能
- **简洁** - 精心挑选的基本功能，直观的界面
- **两种模式** - 快速记录模式和带计时器的自动引导训练模式
- **注重隐私** - 完全离线运行，数据仅保存在您的设备上

## 主要功能

### 训练记录
- 记录日期、时间、组数、次数（或秒数）
- 支持双侧动作（俯卧撑等）和单侧动作（手枪深蹲等）
- 备注功能记录关于动作和见解的笔记

### 查看记录
通过3个标签查看记录：

1. **列表标签** - 列表显示过去的记录，可编辑和删除
2. **图表标签** - 用图表按期间可视化进度（平均/最大/总计）
3. **挑战标签** - 查看目标达成状态，分4个阶段评估

### 挑战设置
- 设置目标组数×目标值（例：3组×50次）
- 通过用前N组的总和判断实现灵活的达成评估
- 进度条颜色根据达成率变化

### 分组管理和收藏
- 将动作分组（例：俯卧撑、深蹲、引体向上等）
- 用分层显示清晰组织
- 用级别（1-10）管理渐进进度
- 用收藏功能在专用分组中显示常用动作

### 备份
- 以JSON格式导出数据
- 支持数据迁移到其他设备或备份
- 以CSV格式导入记录（方便从纸质记录或笔记应用迁移）

### 多语言支持
- 支持中文（简体）、英语、日语、德语、西班牙语和法语
- 根据设备语言设置自动切换

## 使用方法

### 1. 创建动作
从"创建"屏幕添加新动作：

1. 输入动作名称（例："靠墙俯卧撑"）
2. 选择类型（动态：基于次数/静态：基于时间）
3. 选择单双侧（双侧：两侧/单侧：一侧）
4. 选择分组（可选）
5. 设置级别（1-10，可选）
6. 设置挑战（目标组数×目标值，可选）
7. 用★标记收藏（可选）

### 2. 记录训练
从"记录"屏幕添加记录：

1. 选择动作（从收藏或分层显示）
2. 设置组数
3. 输入每组的值
4. 添加备注（可选）
5. 确认日期和时间（必要时更改）
6. 点击"记录"

### 3. 使用训练模式
从"训练"屏幕进行自动引导训练：

1. 选择动作
2. 设置目标组数和次数
3. 计时器设置（每次时长、倒计时、间隔）
4. 点击"开始"
5. 自动倒计时→执行→间隔→下一组
6. 完成后保存记录

### 4. 查看记录
在"查看记录"屏幕查看过去的记录：

- **列表标签**：按训练显示，可编辑和删除
- **图表标签**：用折线图可视化进度
- **挑战标签**：用进度条显示目标达成状态

可按动作和期间筛选。

### 5. 备份数据
从"设置"屏幕导出和导入数据：

**完整备份（JSON）**
- **导出**："导出数据"→选择保存位置
- **导入**："导入数据"→选择JSON文件
  - ⚠️ 导入时删除现有数据

**添加记录（CSV）**
- **下载模板**：用"导出输入模板"获取记录CSV模板
- **导入记录**：用"导入记录"从CSV文件添加记录
  - 添加到现有数据（不删除）
  - 方便从纸质记录或笔记应用迁移

## 挑战判定机制

### 基本规则
通过**前N组的总和**判定是否达到目标。

**例：目标为"2组×20次"时**

**达成模式**：
- 20次+20次+5次→前2组的总和=40次（100%）
- 25次+16次+10次→前2组的总和=41次（102%）

**未达成模式**：
- 15次+15次+15次→前2组的总和=30次（75%）

### 单侧动作的情况
分别计算左右的前N组，用平均值评估。

**例：目标为"2组×20次（每侧）"时**

**两侧达成**：
- 右侧：20次+20次=40次（100%）
- 左侧：19次+21次=40次（100%）
- **平均：100%**→达成

**仅一侧达成**：
- 右侧：20次+20次=40次（100%）
- 左侧：15次+15次=30次（75%）
- **平均：87.5%**→未达成

### 评估标准

进度条颜色根据达成率变化：
- **100%或以上**：完美达成（显示✓达成标记）
- **75-99%**：良好状态
- **50-74%**：差一点
- **0-49%**：加油

## 常见问题

### 问：支持什么训练方法？
答：支持任何自重训练方法，包括Convict Conditioning、StartBodyweight或您自己的自定义程序。可以自由创建动作，因此可以根据您的训练理念使用。

### 问：数据存储在哪里？
答：存储在应用内的本地数据库（SQLite）中。永远不会发送到互联网，完全离线运行。

### 问：会自动备份吗？
答：不会，需要手动导出。我们建议定期备份。

### 问：卸载应用后数据会删除吗？
答：是的，会删除。卸载前务必导出。

### 问：不设置挑战也能使用吗？
答：是的，不设置挑战也可以使用记录功能。不会在挑战标签中显示，但列表和图表可以正常使用。

### 问：支持多语言吗？
答：支持中文（简体）、英语、日语、德语、西班牙语和法语。根据设备语言设置自动切换。

### 问：收藏的动作显示在哪里？
答：收藏的动作自动显示在"收藏"分组中。继续在原分组中显示，因此可以从任一处访问。
